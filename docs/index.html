Strange Toolkit

A Unity editor toolkit for VRChat world creators. Atmosphere presets, smart toggles with persistence, world auditing, and optimization tools.

Add to VCC Download .unitypackage

Requires VRChat Worlds SDK 3.9.0 or higher

Features

Strange Hub

Central manager for atmosphere presets, toggle persistence, and object cleanup tracking.

Smart Toggles

Interactive toggles with GameObject control, animator parameters, material emission, and audio feedback.

Atmosphere Presets

Define multiple skybox, fog, and lighting configurations. Switch between them at runtime.

Editor Dashboard

World management, visual tools, interactables setup, and performance auditing in one place.

World Auditor

Performance scanning, occlusion culling checks, and asset weight analysis for optimization.

Expansions

Modular add-on system for extending toolkit functionality with custom features.

Installation

1 Via VRChat Creator Companion (Recommended)

Click the "Add to VCC" button above, or manually add the repository URL:

https://bellastrangevr.github.io/StrangeToolkit/vpm.json

2 Via UnityPackage

Download the .unitypackage from releases and import it into your Unity project.
